Anatomy of the CSS rules

Subheading 1

CSS trabaja junto con los elementos de html. Son un conjunto de reglas para ver como el contenido de una página debe ser mostrado. Las reglas de CSS contienen dos partes: un selectory una declaración

Subheading 2

Al definir el CSS puede elegir el color de presentación de la página, el tipo de fuente a utilizar, la alineación, el tamaño del párrafo, etc.
